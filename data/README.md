# データファイル置き場

このディレクトリにExcelファイルやCSVファイルをアップロードすると、自動的にdata.jsが生成されます。

## 対応ファイル

- **SAMファイル**: `収益管理・データ用【SAM】.xlsx`
- **スロ天KPIシート**: `スロ天KPIシート.xlsx`
- **Konibetデータ**: `日报数据汇总-データ総和.csv`

## 使い方

1. このディレクトリに最新のExcelファイルをアップロード
2. GitHubにコミット＆プッシュ
3. GitHub Actionsが自動的にdata.jsを生成してプッシュ
4. ダッシュボードが自動更新されます

## 注意事項

- ファイル名は上記のパターンに一致する必要があります
- 複数のファイルを同時にアップロードしても問題ありません
