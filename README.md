# 事業別ダッシュボード

複数事業の収益・KPIを一元管理するダッシュボードです。

## 🚀 デプロイ方法

### GitHub Pages
1. GitHubで新しいリポジトリを作成
2. `index.html`をアップロード
3. Settings → Pages → Source: main branch
4. 数分後に `https://[username].github.io/[repo-name]/` でアクセス可能

### Netlify Drop
1. [Netlify Drop](https://app.netlify.com/drop) にアクセス
2. `index.html`をドラッグ＆ドロップ
3. 即座にURLが発行されます

## 📊 対応事業

| 事業 | データソース | 主なKPI |
|------|-------------|---------|
| 元amuse | Excel収益シート | GGR, NGR, 回収率, LTV |
| DSC | Excel収益シート | GGR, NGR, 回収率, LTV |
| スロ天 | Excel月次シート | GGR, 入出金, RTP |
| AMUSEVIP | CSVエージェント | 収益, ローリング報酬, NGR |
| 広告 | CSV日報 | コスト, CTR, CVR, CPA |
| バカラツール | Excelレポート | 流入, GGR, コスト, NGR |

## 📁 ファイルアップロード

ダッシュボード右上の「アップロード」ボタンから以下のファイルを追加できます：

- **Excel (.xlsx)** - 収益管理シート、スロ天月次シート
- **CSV - エージェント** - AMUSEVIP週別精算
- **CSV - 広告データ** - コスト、インプレッション、クリック
